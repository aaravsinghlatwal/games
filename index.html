🎮 How to Play Bitcoin Miner

💰 Start with 0 Bitcoins. Click the big Bitcoin icon to earn BTC.

🛠️ Use BTC to buy upgrades:

  • Clicker Upgrade – Increases BTC earned per click.
  • Autoclicker Upgrade – Earn BTC automatically every second.

📈 All upgrades scale infinitely using formulas.

🎉 Your BTC can grow to over 65 digits! Become the richest Bitcoin tycoon!


💻 Cheatcodes

  • info – Show game and creator info.
  • avr – Add 1,000,005 BTC instantly.
  • 69 – Set click power to 699 BTC per click.
  • 699 – Set click power to 69 BTC per click.
  • debug – Show full stats (up to 65-digit values).
  • debug_off – Hide the debug overlay.
